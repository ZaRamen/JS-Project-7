Pokemon Guessing Game

Rules:

  1. Guess the Pokémon full name based on the image provided. You can also press enter to do the same action.
  2. 30-second timer for each guess. Try as many times as possible
  3. Click Next Pokémon to go to start guessing a new Pokémon. You can also press Space to do the same action.
  4. Click the hint for a clue on the name. You only get 3 hints. Click the hint button four times to see all the hints. You can also press / to do the same action.
  5. Make sure to do all the key presses when the input field is not focused on. It won't work
  6. Good luck and have fun
